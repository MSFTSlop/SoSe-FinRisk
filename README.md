# SoSe-FinRisk
Fin Risk Project 1

##How to execute this program

Go through the lines step by step (mark the whole while loops; otherwise you may get crashes).

In the first round answer both loops with "n" 
- no stesstest 
- no own of hedge

After going through the main.R once you can restart every step but this time
you answer "y" in the following loop
- own the hedge

The last walk through is answering both loops with "y".
